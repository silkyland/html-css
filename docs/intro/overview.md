# สร้างเว็บร้านกาแฟชุมชนด้วย HTML, CSS และ Git

เวิร์กชอปนี้ออกแบบสำหรับผู้เริ่มต้นที่ยังไม่เคยเขียนเว็บไซต์ เราจะเริ่มจากคำถามว่า Web เกิดขึ้นเพื่อแก้ปัญหาอะไร แล้วค่อยสร้างหน้าเว็บจริงหนึ่งหน้า ตกแต่งให้ใช้ได้ทั้งมือถือและคอมพิวเตอร์ และบันทึกผลงานด้วย Git

## วันนี้เราจะเรียนอะไร

หลักสูตรนี้ครอบคลุมสามชั้นที่ทำงานร่วมกันบน Web:

| ชั้น | ภาษา/เครื่องมือ | หน้าที่ในชิ้นงาน |
|---|---|---|
| โครงสร้างและความหมาย | **HTML** | บอกว่าส่วนไหนเป็นหัวเรื่อง ข้อความ ลิงก์ รูปภาพ |
| การนำเสนอ | **CSS** | กำหนดสี ระยะห่าง การจัดวาง และหน้าจอมือถือ |
| การบันทึกและแชร์ | **Git + GitHub** | เก็บประวัติการแก้ไฟล์ และส่งงานขึ้นออนไลน์ |

เราไม่ได้เรียนทุกอย่างของแต่ละภาษา แต่เลือกเฉพาะสิ่งที่พอสร้างหน้าเว็บร้านกาแฟหนึ่งหน้าให้ครบวงจรในหนึ่งวัน

## ผลลัพธ์เมื่อจบวัน

ผู้เรียนจะสามารถ:

1. อธิบายได้ว่า Web, HTML และ CSS เกิดขึ้นอย่างไร และทำหน้าที่ต่างกันอย่างไร
2. สร้าง `index.html` ที่มีโครงสร้างเชิงความหมายและพื้นฐานการเข้าถึงที่ดี
3. สร้าง `style.css` เพื่อตกแต่งและทำให้หน้าเว็บตอบสนองต่อขนาดหน้าจอ
4. ตรวจปัญหาพื้นฐานด้วย Browser และ HTML Validator
5. ใช้ Git บันทึกงานเป็นลำดับ และส่ง repository ขึ้น GitHub

## ชิ้นงานเดียวที่เราจะต่อยอด

ตลอดวันเราจะทำงานในโฟลเดอร์เดียวชื่อ `my-shop-profile` และพัฒนาเว็บ **“กาแฟบ้านดอย”** ซึ่งเป็นร้านกาแฟชุมชนสมมติ ไม่ต้องสร้างโปรเจกต์ใหม่ในแต่ละบท — ทุกบทต่อจากไฟล์ชุดเดิม

```text
my-shop-profile/
├── index.html
├── style.css
└── images/
    └── community-coffee-shop.webp
```

ช่วงเช้าสร้างโครงสร้างและเนื้อหาด้วย HTML ช่วงบ่ายตกแต่งด้วย CSS แล้วบันทึกงานด้วย Git ท้ายวันจะมีช่วงปรับแบรนด์ให้เป็นชิ้นงานของตนเอง โดยยังใช้โฟลเดอร์และ repository เดิม

::: tip หลักคิดของเวิร์กชอป
ฟังแนวคิดสั้น ๆ → ลงมือแก้ไฟล์ → เปิดดูผลใน Browser → ตรวจ checkpoint  
หากผิด ให้ย้อนดูหัวข้อ “สถานะเริ่มต้น” ของบทนั้น แล้วแก้ให้ตรงก่อนไปต่อ
:::
