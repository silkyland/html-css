# สร้างเว็บร้านกาแฟชุมชนด้วย HTML, CSS และ Git

เวิร์กชอปนี้ออกแบบสำหรับผู้เริ่มต้นที่ยังไม่เคยเขียนเว็บไซต์ เราจะเริ่มจากคำถามว่า Web เกิดขึ้นเพื่อแก้ปัญหาอะไร แล้วค่อยสร้างหน้าเว็บจริงหนึ่งหน้า ตกแต่งให้ใช้ได้ทั้งมือถือและคอมพิวเตอร์ และบันทึกผลงานด้วย Git

## ผลลัพธ์เมื่อจบวัน

ผู้เรียนจะสามารถ:

1. อธิบายได้ว่า Web, HTML และ CSS เกิดขึ้นอย่างไรและทำหน้าที่ต่างกันอย่างไร
2. สร้าง `index.html` ที่มีโครงสร้างเชิงความหมายและพื้นฐานการเข้าถึงที่ดี
3. สร้าง `style.css` เพื่อตกแต่งและทำให้หน้าเว็บตอบสนองต่อขนาดหน้าจอ
4. ตรวจปัญหาพื้นฐานด้วย Browser และ HTML Validator
5. ใช้ Git บันทึกงานเป็นลำดับ และส่ง repository ขึ้น GitHub

## ชิ้นงานเดียวที่เราจะต่อยอด

ตลอดวันเราจะทำงานในโฟลเดอร์เดียวชื่อ `my-shop-profile` และพัฒนาเว็บ **“กาแฟบ้านดอย”** ซึ่งเป็นร้านกาแฟชุมชนสมมติ

```text
my-shop-profile/
├── index.html
├── style.css
└── images/
    └── community-coffee-shop.webp
```

::: tip หลักคิดของเวิร์กชอป
ฟังแนวคิดสั้น ๆ → ลงมือแก้ไฟล์ → เปิดดูผล → ตรวจ checkpoint หากผิดให้ย้อนดู “สถานะเริ่มต้น” ของบทนั้น
:::

## บทหลักและบทเสริม

- **บทหลัก (Core):** เนื้อหาที่ทำร่วมกันในห้องและใช้ตรวจชิ้นงานท้ายวัน
- **บทเสริม (Enrichment):** เนื้อหาสำหรับผู้เรียนที่ทำเสร็จเร็วหรือกลับมาศึกษาต่อ เช่น ตาราง ฟอร์ม CSS Grid และ `git clone`
