# จาก CERN สู่ Web ที่เราใช้วันนี้

**สถานะ: บทหลัก (Core) · เวลาแนะนำ 20 นาที**

## เป้าหมาย

เมื่อจบบทนี้ คุณจะอธิบายได้ว่า Web เกิดจากปัญหาอะไร ใครเขียน HTML รุ่นแรก เหตุใดจึงมี CSS และใครดูแลมาตรฐาน HTML ปัจจุบัน

## สถานะเริ่มต้น

คุณมี `my-shop-profile/index.html` ที่เปิดใน Browser ได้แล้ว บทนี้ยังไม่แก้ไฟล์ แต่จะสร้างภาพรวมก่อนลงมือเขียนโค้ด

## Internet ไม่ใช่ Web

**Internet** คือเครือข่ายที่เชื่อมคอมพิวเตอร์จำนวนมาก ส่วน **World Wide Web** (หรือเรียกสั้น ๆ ว่า Web) เป็นระบบเอกสารและลิงก์ที่ทำงานบน Internet

เปรียบง่าย ๆ: Internet เป็นถนน ส่วน Web เป็นระบบป้ายบอกทางและเอกสารที่เดินทางบนถนนนั้น เราจึงใช้อินเทอร์เน็ตทำอย่างอื่นได้ด้วย เช่น ส่งอีเมล ขณะที่การเปิดหน้าเว็บเป็นการใช้บริการ Web

## จุดเริ่มต้น: ปัญหาการแบ่งปันข้อมูล

ปลายทศวรรษ 1980 นักวิทยาศาสตร์ที่ CERN ใช้คอมพิวเตอร์และระบบเอกสารหลายแบบ การค้นหาและเชื่อมโยงข้อมูลข้ามทีมจึงยาก Tim Berners-Lee เสนอระบบจัดการข้อมูลแบบเชื่อมโยงในปี **1989** แนวคิดนี้ต่อมากลายเป็น World Wide Web

![ภาพวาดบรรยากาศสำนักงาน CERN ราวปี 1990 กับคอมพิวเตอร์ NeXT และแผนผังเอกสารเชื่อมโยง](/images/workshop/web-history-cern.webp)

จุดสำคัญไม่ใช่แค่ “สร้างหน้าเว็บสวย” แต่คือการทำให้เอกสารอ้างถึงกันได้ผ่านลิงก์ และให้คนอื่นเปิดอ่านด้วยโปรแกรมเดียวกันได้

แหล่งข้อมูลหลัก: [CERN — A short history of the Web](https://home.cern/science/computing/the-birth-of-the-web/short-history-web/)

## เส้นเวลาแบบย่อ

| ปี | เหตุการณ์ | ความหมายต่อผู้เรียนวันนี้ |
|---|---|---|
| 1989 | Tim Berners-Lee เสนอแนวคิด Web ที่ CERN | Web เริ่มจากการแก้ปัญหาเชื่อมโยงและแบ่งปันข้อมูล |
| 1990 | Browser/editor, Web server และ HTML รุ่นแรกเริ่มทำงาน | ผู้สร้างเอกสารเขียน HTML แล้ว Browser อ่านและแสดงผล |
| 1993 | CERN เปิดเทคโนโลยี Web ให้ใช้ได้โดยไม่เก็บค่าลิขสิทธิ์ | Web สามารถแพร่หลายและพัฒนาร่วมกันได้ |
| 1994 | ก่อตั้ง W3C; Håkon Wium Lie เริ่มข้อเสนอ CSS | การกำหนดหน้าตาแยกออกจากโครงสร้างเอกสารชัดขึ้น |
| 1996 | CSS1 เป็น W3C Recommendation | CSS มีมาตรฐานร่วมสำหรับการนำเสนอเอกสาร |
| 2004 | WHATWG เริ่มพัฒนาแนวทาง HTML แบบต่อเนื่อง | HTML เติบโตตามการใช้งานจริงบน Web |
| 2019 | W3C และ WHATWG ตกลงพัฒนามาตรฐาน HTML ชุดเดียว | ลดการมีมาตรฐาน HTML สองสายที่ขัดกัน |
| ปัจจุบัน | HTML เป็น **Living Standard** | ภาษาได้รับการปรับปรุงต่อเนื่อง ไม่ต้องรอเลขเวอร์ชันใหญ่ |

แหล่งข้อมูลหลัก: [W3C — History](https://www.w3.org/about/history/), [WHATWG — History of HTML](https://html.spec.whatwg.org/multipage/introduction.html#history)

## ใคร “เขียน HTML”?

Tim Berners-Lee เขียน HTML รุ่นแรก พร้อม Web server และ Browser/editor รุ่นแรก แต่ HTML ที่เราใช้วันนี้ไม่ได้เป็นผลงานของคนคนเดียวอีกต่อไป มาตรฐานปัจจุบันดูแลแบบต่อเนื่องโดย WHATWG และมีผู้พัฒนา Browser กับชุมชน Web ร่วมเสนอและตรวจสอบ

> ในหลักสูตรนี้เราจะเรียกภาษาว่า **HTML** ไม่เรียก “HTML5 เวอร์ชันล่าสุด” เพราะมาตรฐานปัจจุบันเป็น Living Standard และ WHATWG ระบุว่า “HTML5” มักถูกใช้เป็นคำกว้างมากกว่าเลขรุ่นที่หยุดนิ่ง

แหล่งข้อมูลหลัก: [WHATWG — Is this HTML5?](https://html.spec.whatwg.org/multipage/introduction.html#is-this-html5)

## ทำไม HTML เน้น “ความหมาย”

HTML เริ่มจากการอธิบายเอกสารวิทยาศาสตร์ เช่น หัวเรื่อง ย่อหน้า รายการ และการอ้างอิง เมื่อเราเขียน `<h1>` เรากำลังบอกว่า “นี่คือหัวเรื่องหลัก” ไม่ใช่เพียงสั่งให้ตัวอักษรใหญ่

แนวคิดนี้ทำให้ Browser, search engine และเทคโนโลยีช่วยเหลือ (เช่น screen reader) เข้าใจโครงสร้างร่วมกัน จึงเป็นรากฐานของบท Semantic HTML และการเข้าถึงในภายหลัง

แหล่งข้อมูลหลัก: [WHATWG — Background](https://html.spec.whatwg.org/multipage/introduction.html#background)

## แล้วทำไมต้องมี CSS?

ช่วงแรกผู้สร้างเว็บต้องการควบคุมสี ฟอนต์ ระยะห่าง และการจัดหน้า แต่ถ้าเอาคำสั่งหน้าตาปะปนใน HTML เอกสารจะอ่านและดูแลยาก CSS จึงพัฒนาขึ้นเพื่ออธิบายว่าเอกสารที่มีโครงสร้างควรถูกนำเสนออย่างไร

Håkon Wium Lie เริ่มข้อเสนอในปี 1994, Bert Bos ร่วมพัฒนา และ CSS1 ได้เป็น Recommendation ในปี 1996

![เปรียบเทียบบทบาท HTML ในฐานะโครงสร้างเอกสาร กับ CSS ในฐานะการนำเสนอหน้าเว็บ](/images/workshop/html-css-roles.webp)

```text
HTML = โครงสร้างและความหมาย
CSS  = การนำเสนอและการจัดวาง
Browser = อ่านทั้งสองอย่างแล้ววาดเป็นหน้าเว็บ
```

ในเวิร์กชอปนี้เราจะทำตามลำดับเดิมของประวัติศาสตร์เล็กน้อย: สร้างโครงสร้างด้วย HTML ก่อน แล้วค่อยแยกการตกแต่งไปไว้ใน `style.css`

แหล่งข้อมูลหลัก: [W3C — The CSS saga](https://www.w3.org/Style/LieBos2e/history/), [W3C — CSS1 Recommendation](https://www.w3.org/press-releases/1996/css1-rec/)

## ลงมือ

### อธิบายด้วยภาษาของตัวเอง

จับคู่กันแล้วลองตอบโดยไม่เปิดตาราง:

1. Web เกิดมาเพื่อช่วยแก้ปัญหาอะไร?
2. ใครเขียน HTML รุ่นแรก และทำงานที่ใด?
3. HTML กับ CSS รับผิดชอบต่างกันอย่างไร?
4. เหตุใดเราจึงเรียกมาตรฐานปัจจุบันว่า Living Standard?

## ผลที่ควรเห็น

คุณควรเล่าเส้นทางจาก “ปัญหาแบ่งปันข้อมูลที่ CERN” ไปสู่ “เอกสาร HTML ที่ตกแต่งด้วย CSS และเปิดใน Browser” ได้ภายในประมาณหนึ่งนาที โดยไม่สับสนระหว่าง Internet กับ Web

## Checkpoint

- [ ] ไม่สับสนระหว่าง Internet กับ Web
- [ ] บอกชื่อ Tim Berners-Lee และ CERN ได้
- [ ] อธิบายได้ว่า HTML บอกความหมาย ส่วน CSS บอกการนำเสนอ
- [ ] รู้ว่า HTML ปัจจุบันเป็น Living Standard ซึ่ง WHATWG ดูแลต่อเนื่อง
