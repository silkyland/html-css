# รู้จักวิทยากร

## นายบัณฑิต นันทะเทศ

- Senior Full Stack Developer, Data Horizon Co., Ltd.
- ทำงานด้านการพัฒนาเว็บไซต์ทั้งส่วน Front-end และ Back-end
- สนใจการเปลี่ยนโจทย์ทางธุรกิจให้เป็นระบบที่ใช้งานได้จริง

## วิธีเรียนร่วมกันวันนี้

ผู้เรียนไม่จำเป็นต้องจำ tag หรือคำสั่งทั้งหมด สิ่งสำคัญคือรู้ว่า:

1. จะค้นคำตอบจากเอกสารที่น่าเชื่อถืออย่างไร
2. จะอ่านผลบน Browser และข้อความ error อย่างไร
3. จะย้อนกลับไปยัง checkpoint ล่าสุดเมื่อผลไม่ตรงอย่างไร
4. จะอธิบายเหตุผลของโค้ดที่ตัวเองเลือกใช้ได้หรือไม่

::: tip ขอความช่วยเหลือให้แก้ได้เร็ว
เมื่อเรียกผู้สอน ช่วยบอกสามอย่าง: “กำลังทำบทไหน”, “คาดว่าจะเห็นอะไร” และ “ตอนนี้เห็นอะไร” พร้อมเปิดไฟล์กับ Browser ไว้
:::

## ข้อตกลงในห้อง

- ถามได้ทันทีเมื่อไม่เข้าใจ ไม่มีคำถามใดเล็กเกินไปสำหรับผู้เริ่มต้น
- ใช้ข้อมูลร้านและข้อมูลติดต่อสมมติในแบบฝึกหัด
- ไม่คัดลอก token, password หรือข้อมูลส่วนตัวลงใน repository
- ช่วยเพื่อนด้วยการอธิบายแนวคิด ไม่ทำงานแทนทั้งหมด
