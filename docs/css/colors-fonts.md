## หลักจำง่าย

- ใช้สีหลัก 1–2 สี เพื่อให้ภาพรวมสม่ำเสมอ
- ให้สีตัวอักษรตัดกับพื้นหลังและอ่านง่าย
- ใช้ขนาดตัวอักษรสร้างลำดับความสำคัญ
- อย่าใช้ฟอนต์หลายแบบเกินความจำเป็น

:::
