# Tag พื้นฐานสำหรับเนื้อหา

HTML ใช้ **Tag** เพื่อบอกว่าเนื้อหาแต่ละส่วนมีหน้าที่อะไร

<script setup>
const example = {
  html: `<h1>ร้านเบเกอรีหอมกรุ่น</h1>
<h2>เมนูแนะนำ</h2>

<p>ขนมปังอบสดใหม่ทุกเช้า</p>
<p><strong>โปรโมชั่น:</strong> ซื้อ 2 ชิ้น ฟรี 1 ชิ้น</p>

<hr>

<p>เปิดทุกวัน<br>เวลา 08:00–18:00 น.</p>`
}
</script>

## Tag ที่ใช้บ่อย

| Tag | หน้าที่ | ตัวอย่าง |
|-----|---------|---------|
| `<h1>` – `<h6>` | สร้างหัวข้อตามลำดับความสำคัญ | `<h1>ชื่อร้าน</h1>` |
| `<p>` | สร้างย่อหน้าของเนื้อหา | `<p>ขนมปังอบสดใหม่</p>` |
| `<strong>` | เน้นข้อความสำคัญ (ตัวหนา) | `<strong>โปรโมชั่น</strong>` |
| `<br>` | ขึ้นบรรทัดใหม่ | `เปิดทุกวัน<br>08:00–18:00` |
| `<hr>` | สร้างเส้นคั่นแนวนอน | `<hr>` |

::: tip
เลือก Tag จาก “ความหมาย” ของเนื้อหา ไม่ใช่เพียงรูปลักษณ์
:::
