# รายการและตารางข้อมูลธุรกิจ

## รายการสินค้า

<script setup>
const example = {
  html: `<h2>บริการของเรา</h2>
<ul>
  <li>ออกแบบนามบัตร</li>
  <li>ออกแบบโปสเตอร์</li>
  <li>ดูแลเพจธุรกิจ</li>
</ul>

<h2>ตารางราคา</h2>
<table>
  <tr>
    <th>แพ็กเกจ</th>
    <th>ราคา</th>
  </tr>
  <tr>
    <td>Starter</td>
    <td>1,500 บาท</td>
  </tr>
</table>`
}
</script>

<LivePreview v-bind="example" height="260px" />

- `<ul>` คือรายการแบบไม่มีลำดับ
- `<ol>` คือรายการแบบมีลำดับ
- ตารางเหมาะกับข้อมูลที่ต้องเปรียบเทียบเป็นแถวและคอลัมน์

::: details ภาพประกอบที่แนะนำ
ผลลัพธ์ของรายการบริการและตารางราคาใน Browser
:::
