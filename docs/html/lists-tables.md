# บทเสริม: รายการและตารางข้อมูล

**สถานะ: บทเสริม (Enrichment) · เวลาแนะนำ 20 นาที**

บทนี้ไม่ใช่เงื่อนไขของชิ้นงานหลัก เหมาะสำหรับผู้เรียนที่ผ่าน checkpoint Accessibility แล้ว

## เป้าหมาย

เลือก `<ul>`/`<ol>` ตามความสำคัญของลำดับ และสร้างตารางเวลาชงกาแฟที่มี caption กับหัวตารางชัดเจน

## สถานะเริ่มต้น

ใช้ `index.html` ร้านกาแฟบ้านดอยฉบับ semantic จากบทหลัก

## รายการ: ลำดับสำคัญหรือไม่?

```html
<h2>บริการของเรา</h2>
<ul>
  <li>เมล็ดกาแฟคั่ว</li>
  <li>เครื่องดื่มหน้าร้าน</li>
  <li>เวิร์กชอปดริปกาแฟ</li>
</ul>

<h2>วิธีสั่งเมล็ดกาแฟ</h2>
<ol>
  <li>เลือกชนิดและระดับการคั่ว</li>
  <li>แจ้งรูปแบบการบด</li>
  <li>ยืนยันที่อยู่และชำระเงิน</li>
</ol>
```

ใช้ `<ol>` เมื่อการสลับลำดับทำให้ความหมายหรือผลลัพธ์เปลี่ยน ใช้ `<ul>` เมื่อเป็นกลุ่มรายการที่ไม่มีลำดับตายตัว

## ตาราง: ใช้กับข้อมูล ไม่ใช่จัดหน้า

เพิ่มใน `<main>`:

```html
<section aria-labelledby="workshop-title">
  <h2 id="workshop-title">รอบเวิร์กชอปดริปกาแฟ</h2>
  <table>
    <caption>ตารางรอบวันเสาร์ ราคา 350 บาทต่อคน</caption>
    <thead>
      <tr>
        <th scope="col">รอบ</th>
        <th scope="col">เวลา</th>
        <th scope="col">จำนวนที่ว่าง</th>
      </tr>
    </thead>
    <tbody>
      <tr>
        <th scope="row">รอบเช้า</th>
        <td>09:30–11:00</td>
        <td>4 คน</td>
      </tr>
      <tr>
        <th scope="row">รอบบ่าย</th>
        <td>13:30–15:00</td>
        <td>2 คน</td>
      </tr>
    </tbody>
  </table>
</section>
```

`caption` บอกว่าตารางนี้เกี่ยวกับอะไร ส่วน `scope` เชื่อมหัวตารางกับข้อมูลในแถวหรือคอลัมน์ ไม่ควรใช้ table เพื่อจัดตำแหน่งภาพหรือกล่องสินค้า

## ผลที่ควรเห็น

Browser แสดงรายการแบบจุด รายการแบบตัวเลข และตารางสองแถว แม้ยังไม่สวยแต่ความสัมพันธ์ของข้อมูลอยู่ใน HTML แล้ว

## Checkpoint

- [ ] ขั้นตอนที่ต้องเรียงลำดับใช้ `<ol>`
- [ ] ตารางมี `<caption>` ที่เข้าใจได้โดยไม่ต้องอ่านย่อหน้ารอบข้าง
- [ ] หัวคอลัมน์ใช้ `scope="col"` และหัวแถวใช้ `scope="row"`
- [ ] ไม่มี table ที่สร้างขึ้นเพื่อจัด layout

## บันทึกแหล่งข้อมูล

โครงสร้างตารางที่เข้าถึงได้อ้างจาก [W3C WAI — Tables Tutorial](https://www.w3.org/WAI/tutorials/tables/)
