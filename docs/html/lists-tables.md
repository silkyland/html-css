## Tag สำหรับรายการและตาราง

| Tag | ประเภท | หน้าที่ |
|-----|--------|---------|
| `<ul>` | รายการไม่มีลำดับ | แสดงรายการแบบจุดนำ |
| `<ol>` | รายการมีลำดับ | แสดงรายการแบบมีเลขลำดับ |
| `<li>` | รายการย่อย | แต่ละไอเทมใน `<ul>` หรือ `<ol>` |
| `<table>` | ตาราง | แสดงข้อมูลเปรียบเทียบเป็นแถวและคอลัมน์ |
| `<tr>` | แถว | แถวหนึ่งของตาราง |
| `<th>` | หัวคอลัมน์ | เซลล์หัวข้อ (ตัวหนา จัดกลาง) |
| `<td>` | เซลล์ข้อมูล | เซลล์ข้อมูลทั่วไป |

::: tip เลือกประเภทรายการให้ถูก
ใช้ `<ol>` เมื่อลำดับสำคัญ (เช่น ขั้นตอน) และ `<ul>` เมื่อลำดับไม่สำคัญ (เช่น รายการสินค้า)
:::

::: details ภาพประกอบที่แนะนำ
ผลลัพธ์ของรายการบริการและตารางราคาใน Browser
:::
